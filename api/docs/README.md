# API Documentation

This directory contains documentation for the REST API. You can find OpenAPI/Swagger definitions, Postman collections, or other API documentation tools here.

## Endpoints

### Users

#### Get all users

```
GET /api/users
```

#### Get a specific user

```
GET /api/users/:id
```

#### Create a new user

```
POST /api/users
```

#### Update a user

```
PUT /api/users/:id
```

#### Delete a user

```
DELETE /api/users/:id
```
